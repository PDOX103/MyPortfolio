# 3D Integrated Portfolio

A polished React + Vite portfolio upgraded with a CSS-powered 3D hero, glassmorphism cards, responsive sections, animated skills, project showcase, process timeline, and contact CTA.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Edit your content

Open `src/App.jsx` and update:

- `profile` for your name, role, location, email, and resume link
- `projects` for your real project list
- `skills` for your stack
- `process` and `services` for your workflow and offerings

The 3D design is built with React and CSS only, so no extra 3D library is required.
